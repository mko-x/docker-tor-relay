# docker-tor-relay

## info

Easily run a tor relay node in docker.

Find a docker-compose sample configuration.

## environment

### TARGET_TYPE

Default: middle
Options: middle, exit, bridge

### RELAY_PORT

Default: 9001

### RELAY_BANDWIDTH_RATE

Default: 400000 KBytes

### RELAY_BANDWIDTH_BURST

Default: 1000000 KBytes

### CONTACT_NAME

Default: m-ko

### CONTACT_EMAIL

Default: dev@m-ko.de
